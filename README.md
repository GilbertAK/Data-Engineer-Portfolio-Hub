* **Machine Learning:** Predictive analytics for energy and fintech.
* **Data Science:** Real-time business dashboards.

## 🛠️ Technical Stack
* **Languages:** Python (Expert), JavaScript, HTML5/CSS3.
* **Data Science:** Pandas, Numpy, Scikit-Learn, Matplotlib.
* **Automation:** Requests, BeautifulSoup4, Selenium.
* **Deployment:** Streamlit, GitHub Actions.

## 📂 Project Structure
* `app.py`: Main entry point (Profile & Professional Story).
* `pages/`: Specialized project modules (Interactive Demos).
* `assets/`: Professional branding and visual resources.
* `data/`: Processed datasets and CSV reports.

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/GilbertAK/Data-Engineer-Portfolio-Hub.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Launch the app: `streamlit run app.py`
